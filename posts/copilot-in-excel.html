Copilot in Excel

The current integration of M365 Copilot into MS Excel is… messy. There are multiple ways to interact with Copilot, and it’s not always clear which one you should be using. This post is my attempt at understanding and documenting the various ways that M365 Copilot works with Excel.

Note: This information is likely to be out of date soon after it's published, so be aware that the situation described below might no longer be current.

Introduction

I recently gave a presentation on how M365 Copilot integrates with Excel, and I thought it might be useful to share some of what I've learnt, not only because “sharing is caring”, but also because the situation was far more complicated than I had initially assumed.

App Skills in Excel

This is available by selecting the drop-down arrow on the Copilot button (which is located on the far right of the Home ribbon) and then selecting App skills.

Once you've selected App Skills, you'll be presented with a side-panel on the right-hand side of the screen. This side-panel will contain a list of available App Skills, and you can select the one you want to use by clicking on it.

The App Skills in Excel option is the only one (currently) that will edit the workbook directly using Excel's built in formulas and functions. But, be aware, one of the listed app skills is NOT like the others.

App Skills in Excel (Give a unique insight using Python)

The option to "Give a unique insight using Python" will NOT edit your workbook using Excel's built in functions and formulas, instead it will use Python to analyse your workbook and then present the results in a side-panel.

Why this option is in this list, and not a separate item in the Copilot dropdown list is beyond me.

The Python code that's generated is not editable, and you can't use it to edit your workbook. The only thing you can do with this code is copy it to the clipboard, and then paste it into a Python environment (such as Visual Studio Code) to run it.

Even though the code can't be edited, and even though it's not using Excel's functions and formulas, it can still be quite useful. For example, the Python chart generated above was based on a worksheet that contained a year’s worth of commercial aircraft C- and Heavy-check maintenance events. The AI "reasoned" that plotting the maintenance event duration against aircraft age would be useful (which it was). It even added trendlines so that I could easily see the relationship between maintenance event duration and aircraft age.

App Skills in Excel (Advanced Analysis)

By simply entering the following prompt: "Advanced analysis" you can trigger the Advanced Analysis mode of Copilot. This mode is similar to the "Give a unique insight using Python" mode, in that it will analyse your workbook using Python. The difference is that the results are presented in a new Python enabled Worksheet. And you have the option to edit the Python code that is generated.

After enabling Advanced Analysis mode, I then asked Copilot to plot the maintenance event duration distributions for C-checks only, for the A320 and 737 families of aircraft, using Violin plots. I picked Violin plots specifically as they are not a standard chart type in Excel, also, they look like Manta rays, which is kinda cool.

Chat in Excel (default)

As Chat is now the default, all you need to do is to click the Copilot icon, but it is also shown as an option in the dropdown.
