Examples

The following examples have not been configured specifically to demonstrate the effects of CSS Balance, they are simply the result of enabling text-wrap: balance; on this website.

Figure 1 below shows the first 3 cards from my homepage. Figure 2 shows the same 3 cards but with text-wrap: balance; enabled.

As you can see, CSS Balance has had an effect on all 3 cards from the first row. In total, enabling CSS Balance altered the titles of 13 out of 22 card titles on my homepage. This is not entirely surprising, as CSS Balance will only alter a title if that title requires more than a single line, and this is more likely where width is limited, such as in a typical card design. I wouldn't expect it to have an effect on that many titles on article pages (at least not on desktop).

Where CSS Balance is probably most useful is in stopping scenarios like that show below in Figure 3, i.e where a single number has appeared on a line by itself.
